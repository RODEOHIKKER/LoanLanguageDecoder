# LoanLanguageDecoder
Loan Language Decoder is a web application that makes any loan agreement comprehensible to anyone, regardless of financial or legal literacy. A user pastes or uploads their loan document — a microfinance agreement, personal loan contract, or NBFC terms sheet. The application sends this to an LLM, which identifies every significant clause, explains it in plain language anyone can understand, and classifies it by risk level. Results appear as color-coded cards: green for standard terms, yellow for clauses worth reading carefully, and red for potentially predatory or high-risk terms such as compounding penalty clauses, hidden processing fees, or lender-discretion prepayment charges. Each card shows the original clause alongside its plain-language explanation. Users can toggle output into major Indian regional languages for vernacular accessibility. No legal background required to use it. No sign-up, no cost to the borrower — just clarity at the moment it matters most.
